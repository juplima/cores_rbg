# Atividade - Padrões de Cores RGB

Projeto desenvolvido para a disciplina Computador e UX.

## Conteúdo da atividade
- Criação de cores utilizando o sistema RGB
- Desenvolvimento de degradês
- Criação de padrões visuais
- Construção de uma paleta de cores para interfaces digitais

## Tecnologias utilizadas
- HTML
- CSS (cores RGB)
